Bits & Bobs

Some years ago I created a virtual instrument using a Steinway piano I had available to me. If you happen to use the DAW Reason, in this zip file you'll find an .sxt instrument compatible with Reason, along with all the 416 individual clip recordings of a piano from my hometown.
