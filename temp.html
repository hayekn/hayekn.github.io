Include this .cls file in the folder containing your latex document, and define your document class as \documentclass[notoc/toc]{lecturenew} (pick either notoc or toc to specify whether you want a table of contents). Your preamble should look like

\pagenumbering{gobble}
\title{Party Rockers in the House Tonight}
\subtitle{ft. Martha Stewart}
\speaker{Me}
\author{God}

with \title and \speaker manditory. When you compile, your definition for \title will appear in the header on even pages, on the left side. To define a header for odd-pages (appearing on the right), use the following command:

\renewcommand\partcontent{Current Section}

This will define the header for all subsequent pages until it is redefined (or eliminated, by using an empty argument). You can define page numbers, too, with

\pagenumbering{arabic}
\setcounter{page}{1}

wherever you first want page numbers to start. They will appear in the header opposite the side where any text might also appear. You can reuse the \pagenumbering{gobble} command from above to clear page numbers.

This class is preloaded with a lot, and I mean a lot, of packages and shorthand commands I find useful. Among the fun ones are: xfrac, which gets you some nice slant fraction action, cancel, which can draw lines through expressions (with arrows even), epigraph, which will paste a spot below the TOC to put a quote from your favorite professor, and stmaryrd, or St. Mary's Road, which has a bunch of cooler-than-normal looking brackets and such. As for framing, I've defined the mdframed style theorem to be a green bubble with some borders, and this can be edited manually. To use: \begin{mdframed}[style=theorem] ... \end{mdframed}

Finally, the page content will shift from left to right as you go along, and the extra real estate created by this will accomodate some margin text, which can be inserted using \margintext{}
